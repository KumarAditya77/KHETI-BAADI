# KHETI-BAADI 🌱

An AI-Powered Plant Disease Detection and Organic Farming Assistance Platform.

## Overview

KHETI-BAADI is a modern web platform that helps farmers and gardening enthusiasts identify plant diseases using artificial intelligence and provides organic farming solutions. The platform is designed to be user-friendly, responsive, and accessible on all devices.

## Features

- 🌿 AI-powered plant disease detection
- 🌱 Organic farming recommendations
- 📱 Responsive design for all devices
- 📚 Knowledge base for plant care
- 📞 Contact form for support

## Technologies Used

- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome for icons
- Google Fonts (via CDN)
- Vanilla JavaScript for interactivity
- CSS Grid and Flexbox for layout
- CSS Animations for enhanced user experience

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/kheti-baadi.git
   cd kheti-baadi
   ```

2. **Open in Browser**
   - Simply open the `index.html` file in your preferred web browser
   - Or use a local server like Live Server in VS Code

3. **Customize**
   - Replace placeholder content in `index.html` with your own content
   - Modify colors and styling in `css/style.css`
   - Add more functionality in `js/script.js`
